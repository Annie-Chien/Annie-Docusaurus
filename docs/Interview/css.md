## CSS 偽類 vs 偽元素{#q1}

| 偽類（pseudo-class）     | 偽元素（pseudo-element） |
| ------------------------ | ------------------------ |
| 依據元素的特殊狀態來選取 | 元素的特定部分           |
| 單一冒號                 | 雙冒號                   |
| `:hover`, `:focus`       | `::before`, `::after`    |
